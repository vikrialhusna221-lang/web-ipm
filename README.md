## ✨ Tentang Proyek
Website **[ipmsmamuhi.my.id](https://ipmsmamuhi.my.id)** adalah jantung digital dari Pimpinan Ranting Ikatan Pelajar Muhammadiyah SMA Muhammadiyah 1 Metro. Platform ini dirancang untuk menyatukan kreativitas, informasi, dan administrasi dalam satu wadah yang modern.

## 🚀 Fitur Unggulan
* **⚡ Dashboard Admin Cerdas:** Navigasi sidebar yang modular, memisahkan konten secara rapi baik di Desktop maupun Mobile.
* **📱 Mobile First Design:** Optimasi viewport yang memastikan tidak ada elemen yang terpotong di layar HP.
* **📰 Media Informasi:** Pusat berita dan dokumentasi kegiatan ranting yang *up-to-date*.
* **🔒 Secure Authentication:** Sistem login pengurus yang aman dan efisien.

---

## 🛠️ Tech Stack & Perkakas
Proyek ini dibangun dengan dedikasi menggunakan:

* **Frontend:** `HTML5`, `CSS3`, `JavaScript` (Modern UI/UX)
* **Backend:** `PHP` / `Laravel` (Powering the Logic)
* **Database:** `MySQL` (Reliable Data Storage)
* **Server:** `Shared Hosting` / `Nginx`

## 🔧 Panduan Instalasi Cepat
Ingin mencoba di lokal? Cukup ikuti 3 langkah ini:

1.  **Clone & Masuk**
    ```bash
    git clone [https://github.com/username/ipm-smamuhi.git](https://github.com/username/ipm-smamuhi.git) && cd ipm-smamuhi
    ```
2.  **Siapkan Database**
    Import `database.sql` ke MySQL Anda dan sesuaikan konfigurasi di file koneksi.
3.  **Buka di Browser**
    Akses melalui `localhost/ipm-smamuhi` dan mulai beraksi!
